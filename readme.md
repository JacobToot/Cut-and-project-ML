This repository contains code that has been written for my MSc Research Project in Physics at the Rijksuniversiteit Groningen (University of Groningen) in 2025/2026. The code includes: 
- cut-and-project simulations 
- de Bruijn's Multigrid method for quasiperiodic tilings
- other simple simulations for dataset generation
- utility functions for working with cut-and-project simulations
- ML architectures including best weights found during thesis research for particular use cases 
- an algorithmic way to extract parameters from cut-and-project sequences
- plotting helpers
- a number of example scripts that apply the best weights and reproduce graphs that appear in the thesis
